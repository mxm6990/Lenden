# Lenden — Product Scope v1

**Brand:** লেন্দেন (Lenden)  
**Tagline:** Clarity in every investment.

## v1 status

Lenden v1 is a **mock / prototype** mobile-first app for DSE retail investing UX.

## In scope

- **Home** — Portfolio summary, buying power, DSE status, holdings preview, watchlist, past transactions
- **Market** — DSE index, search, stock list
- **Portfolio** — Holdings, chart, industry allocation detail
- **Profile / Trust Center** — KYC, linked accounts, security, legal, help, risk disclosure, support tickets
- **Stock detail** — Quotes, your position, about
- **Mock buy flow** — Preview and mock submit only

## Out of scope (v1)

- Real trading or order routing
- Real payment movement (bKash/bank transfers)
- Investment advice
- Live market data feeds
- **Learn tab** — built but disabled for later rollout
- **Quick actions** — disabled for later rollout

## Technical stack

React 19, TypeScript, Vite, Tailwind v4, Capacitor 7 (iOS shell)

## Compliance stance

Prototype environment. Mock trading only. Not financial advice. Securities services require licensed brokerage partnerships and regulatory approval.
